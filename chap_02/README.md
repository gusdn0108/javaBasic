- 4. 증감 연산 (++, --)

1. ++

- 1-1 ++앞에 있는경우

ex)

System.out.println(val); // 10
System.out.println(++val); // 11
System.out.println(val); // 11

명령어가 읽힌 시점에서 ++

- 1-2 ++뒤에 있는경우
  val = 10;
  System.out.println(val); // 10
  System.out.println(val++); // 10
  System.out.println(val); // 11

명령어가 다음에 또 발생했을때 ++

- 증감 연산같은 경우 좀 많이 착각 할 수있으니 주의
